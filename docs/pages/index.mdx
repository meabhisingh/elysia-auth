# @auth/elysia

**@auth/elysia** is a powerful authentication middleware for [Elysia.js](https://elysiajs.com) applications, built on top of [`@auth/core`](https://authjs.dev). It provides seamless integration with various authentication providers and robust middleware solutions for protecting your routes.

## Quick Start

```bash
bun add @auth/elysia
```

Basic usage:

```typescript
import { Elysia } from 'elysia'
import { ElysiaAuth, authGuard } from '@auth/elysia'

const app = new Elysia()
  .use(ElysiaAuth({
    secret: process.env.AUTH_SECRET,
    providers: [
      // Your auth providers
    ]
  }))
  .group('/admin', app => app
    .use(authGuard())
    .get('/dashboard', () => 'Protected Dashboard')
  )
  .listen(3000)
```
